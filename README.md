# BFInterpreter - a CLI Brainfuck Interpreter
## Because that's a thing the world needs

A simple python-based BrainFuck (BF) interpreter I made in an hour over Christmas while bored.
Should probably come back and do in a more optimised language sometime.

Usage: 
* Call bfi.exe from your console of choice with the file path to the target program
* Alternatively if you'd rather use the raw source code, `<command for your chosen python interpreter> bfi.py <file_path>`

.txt and .bf file formats are supported.
